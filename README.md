# justcompanydata

Instructions (*nix): 
1. git clone https://github.com/flowejam/justcompanydata.git
2. cd justcompanydata/project/
3. mvn package
4. java -jar target/project-0.0.1-SNAPSHOT.jar 
5. Open your browser and enter http://localhost:8080/
